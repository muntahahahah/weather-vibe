# 🌦️ Weather Vibe

A beautiful, anime-inspired weather app that changes its background based on real-time weather conditions!

## ✨ Features

- 🎨 **Dynamic anime backgrounds** that match the weather — rain, snow, sun, clouds, fog, and more!
- 🌍 **Real-time weather data** powered by OpenWeatherMap API
- 📱 **Fully responsive** — looks stunning on mobile and desktop
- 💜 **Pastel glassmorphism UI** with smooth animations
- 🌙 **Day & night detection** — different vibes for different times
- ⚡ **Instant search** — just type a city and feel the vibe

## 🎯 How It Works

1. Type any city name in the search box
2. The app fetches live weather data from OpenWeatherMap
3. Based on the weather condition (rain, sun, snow, etc.) and time of day, the background automatically switches to a matching anime-style scene
4. Enjoy the vibe! ✨

| Weather | Background Vibe |

| 🌫️ Fog / Mist | Mysterious foggy forest |

## 📱 Responsive Design

Weather Vibe is built to look great on:

- 💻 **Laptops & Desktops** — centered layout, full experience
- 📱 **Mobile Phones** — compact card, touch-friendly, no scrolling needed
- 📲 **Tablets** — adaptive sizing for all screen sizes

## 🎨 Design Highlights

- **Glassmorphism cards** with translucent backgrounds
- **Smooth theme transitions** when weather changes
- **Custom pastel color palettes** for each weather type
- **Text shadows & overlays** for readability on any background
- **Animated elements** — floating emoji, spinning loader, slide-up cards

## ⚠️ Note

This project uses a demo API key for educational purposes. The key is visible in the frontend code as this is a static site. For production use, implement a backend proxy to secure API keys.
